How To Put Movies On Your PlayStation Portable

Because apparently you can’t just drag and drop them…

What you Need

  • Your .mp4 file
  • Handbrake
  • This preset file (Original Source)

Guide

  1. Download and install Handbrake
  2. Open Handbrake
    • You may need an additional .net framework installed, make sure to install that when it asks
  3. Import your folder of .mp4s or singular .mp4 on the “Source Selection” screen
  4. Once that has loaded, click Presets > Import from File on the menu bar
  5. Import the psp.json saved from above
  6. Look on the menu bar again and click Tools > Preferences
  7. Click the Output Files option on the left menu
  8. Change the default file path and file format how you’d like
    • This changes where the process files go on your computer and the way they are named respectively
  9. Look back at the main application window
  10. Change the preset field from the default to PSP
  11. Look in the dropdown for Custom Presets > PSP
  12. Look at the action bar and click Add All
  13. Click on the Queue button and review your settings
    • Make sure the Preset field says PSP or the files will be converted incorrectly
  14. Click Start queue
  15. Wait for your videos to encode and you’re done!
